### Install
pip install -r requirements.txt


### Create an .env file with

```
YOUR_REFRESH_TOKEN=
ORG_ID=
```

Update them with your account information.

ORG_ID can be taken from the url, when you login to your account for eg. https://app.hubstaff.com/organizations/XXXXXX/insights

And REFRESH_TOKEN can be generated by visting the https://developer.hubstaff.com/personal_access_tokens, create PAT(Personal Access Token) and use that as REFRESH_TOKEN.

And then run the summary.py file by 

`python summary.py`
